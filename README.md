# css-generator-ia
Ferramenta web para geração, pré-visualização em tempo real e cópia de blocos de código CSS. Desenvolvido com HTML5, CSS3 e JavaScript, demonstrando domínio em manipulação dinâmica de interface (DOM) e foco em produtividade para desenvolvedores.

Como rodar o projeto localmente

Clone este repositório.

Obtenha uma chave gratuita em [Groq Cloud](https://console.groq.com/home).

No arquivo scripts.js, insira sua chave na variável chaveApi.

Abra o index.html no seu navegador.
